THE INLINE ELEMENT ROUNDUP

em element : within a sentence for stressed emphasis,nearly always displays italics by default.

strong element : within a sentence for strong importance, displays bold by default.

b element : used for bold.

i element : used for italics.

s element : used to strike through.

u element : used to underline.

small element : used to make the sentence smaller type size.

q element : used for short inline quotation.

abbr element : used for abbreviation or acronym

cite element : used for citation (reference to another document), by default rendered in italics text.

dfn element : used for defining terms, can be formatted using style sheets.

program code elements :

code
variable
program sample
user-entered keyboard strokes

sub element : used for subscript, example : H2O

sup element : used for superscript, example : E=MC2

mark element : used for highlighted text.
