ORGANIZING PAGE CONTENT

The page discusses about elements that are used for grouping several small chunks inside these organizing elements

The use of main element :
to identify the primary content of a page or application.
pages should have only one main section.
it should not be nested within an article, aside, header,footer, or nav.

The use of section element :

  • to separate sections of a large doc,thematic section
  • separated part for easy navigation

The use of article element :

  • used for self contained work
  • could be thought of as special 'section'
  • useful for newspaper article, blog post etc.
  • a large article could be broken into smaller sections and vice versa

The use of aside element :

  • something which is directly not related to the document,could be used for background info, links etc.

  • resources :

  • Typekit
  • Google Fonts

The use of nav element :

  • used for links for primary navigation around the site
  • various sections could be navigated using the links like this
